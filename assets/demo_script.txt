# AI Desktop Assistant - Demo Script (90 seconds)

## Pre-Demo Setup
1. Ensure Ollama is running: `ollama serve`
2. Open terminal in project folder
3. Have Chrome/browser closed initially
4. Position screen for recording (1920x1080 recommended)

## FFmpeg Commands for GIF Creation

```bash
# Record screen (use OBS or ShareX)
# Then convert to GIF:

# demo_voice.gif (15 seconds, starting at 20s)
ffmpeg -ss 00:00:20 -t 15 -i demo_full.mp4 -vf "fps=10,scale=800:-1:flags=lanczos" -c:v gif demo_voice.gif

# demo_browser.gif (15 seconds, starting at 35s)
ffmpeg -ss 00:00:35 -t 15 -i demo_full.mp4 -vf "fps=10,scale=800:-1:flags=lanczos" -c:v gif demo_browser.gif

# demo_notepad.gif (15 seconds, starting at 50s)
ffmpeg -ss 00:00:50 -t 15 -i demo_full.mp4 -vf "fps=10,scale=800:-1:flags=lanczos" -c:v gif demo_notepad.gif
```

## Tips for Best Recording

1. Use a quiet room (Whisper works better)
2. Speak clearly and at normal pace
3. Wait for each action to complete before speaking again
4. Keep mouse movements slow and deliberate
5. Use 1920x1080 resolution for crisp GIFs
6. Close unnecessary applications to avoid distractions
